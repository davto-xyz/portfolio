# Xtart FP - Frontend Developer & Data Analyst

En **Xtart FP (anteriormente FP Claudio Galeno)**, trabajé en el desarrollo de plataformas educativas y sistemas de gestión, combinando desarrollo frontend con análisis de datos para mejorar la experiencia de aprendizaje.

## Funciones principales desarrolladas

### Plataforma de Aprendizaje (Moodle)
- **Personalización de temas**: Desarrollo de interfaces modernas para Moodle
- **Módulos personalizados**: Creación de plugins específicos para necesidades educativas
- **Integración de contenidos**: Sistema de gestión de materiales multimedia
- **Dashboard de progreso**: Paneles interactivos para seguimiento de estudiantes

### Sistema de Gestión de Pagos
- **Interface de usuario**: Desarrollo del frontend para procesamiento de pagos
- **Validación de formularios**: Implementación de validaciones client-side y server-side
- **Integración con pasarelas**: Conexión con múltiples proveedores de pago
- **Reportes financieros**: Dashboards para análisis de transacciones

### Análisis de Datos Educativos
- **Métricas de engagement**: Análisis de participación de estudiantes
- **Reportes de rendimiento**: Generación automática de informes académicos
- **Dashboards interactivos**: Visualización de datos para administradores
- **Optimización de contenidos**: Análisis de eficacia de materiales educativos

## Tecnologías implementadas

- **PHP & Laravel**: Backend robusto para sistemas de gestión
- **Moodle**: Personalización y desarrollo de plugins educativos
- **JavaScript**: Interactividad y validaciones frontend
- **MySQL**: Gestión de bases de datos educativas y financieras
- **HTML5/CSS3**: Interfaces modernas y responsive

## Impacto conseguido

- **40% mejora** en la experiencia de usuario de la plataforma Moodle
- **Automatización** del 70% de procesos de gestión de pagos
- **Reducción del 50%** en tiempo de generación de reportes
- **Implementación** de sistema de análisis de datos en tiempo real

Esta experiencia me permitió especializarme en el desarrollo de plataformas educativas y adquirir habilidades en análisis de datos aplicado al sector formativo.
